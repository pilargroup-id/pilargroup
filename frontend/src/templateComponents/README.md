# Template Components

Folder ini berisi salinan mandiri untuk `Header` dan `Sidebar`.

## Isi

- `Header.jsx`
- `Sidebar.jsx`
- `templateComponents.css`
- `navigation.js`
- `accessControl.js`
- `departmentFilter.js`
- `authStorage.js`
- `logoutService.js`
- `assets/`

## Cara pakai

```jsx
import { Header, Sidebar } from '@/templateComponents'
import '@/templateComponents/templateComponents.css'
```

Kalau mau dipakai di project lain, cukup pindahkan folder ini bersama asset dan CSS yang ada di dalamnya.
